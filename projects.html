Project 1: Adventure Game - Treasure Hunter

About this Project

Treasure Hunter is an interactive adventure game that combines classic game mechanics with creative coding. Players navigate through a dynamic environment, collecting treasures while avoiding obstacles and challenges. The game features smooth collision detection, responsive controls, and engaging visual feedback.

I created this project to explore fundamental game development concepts through p5.js, focusing on player movement, collision systems, and game state management. The challenge was balancing accessibility for new players while maintaining engaging gameplay mechanics.

Key Features

  • Real-time collision detection
  • Dynamic obstacle generation
  • Score tracking system
  • Responsive keyboard controls
  • Progressive difficulty scaling

Technologies Used

  • p5.js
  • JavaScript
  • HTML5 Canvas

Project 2: Interactive Experience - Steamboat Willie Restoration

About this Project

This interactive experience reimagines the historic Steamboat Willie animation through the lens of creative coding and physics simulation. Using Matter.js for realistic physics interactions and p5.js for visual rendering, users can interact with restored elements from the public domain classic, bringing new life to vintage animation.

The project explores the intersection of digital preservation and creative technology. I wanted to demonstrate how modern web technologies can make historical content interactive and engaging for contemporary audiences while respecting the original artistic vision. The physics interactions create playful moments that invite exploration and discovery.

Key Features

  • Physics-based interactions with Matter.js
  • Frame-by-frame animation restoration
  • Interactive scene elements
  • Responsive particle systems
  • Audio-visual synchronization

Technologies Used

  • p5.js
  • Matter.js
  • JavaScript
  • HTML5 Canvas

Project 3D: Solarpunk 3D Experience

About this Project

Voltage Village is an interactive 3D exploration game that combines WebVR technology with generative art to create an engaging solarpunk-themed experience. Built with A-Frame and p5.js, players navigate a sustainable village while collecting energy cores to power the community. The game features dynamic audio visualizations, animated characters, holographic displays, and procedurally generated mathematical art pieces scattered throughout the village.

Players can interact with the environment using keyboard controls or voice commands, discovering music-reactive elements and cultural exhibits as they explore. With support for both desktop browsers and VR headsets, Voltage Village demonstrates the creative possibilities of combining web-based 3D frameworks with creative coding libraries to build immersive, mission-driven experiences.

Why 3D?

This project fundamentally requires 3D to achieve its goals. A static image or video would fail to capture the essential interactive and spatial nature of the exploration experience. The ability to move through the village, navigate between buildings, and interact with objects from different angles creates a sense of presence and agency that's impossible in 2D media.

The gameplay mechanics rely on spatial awareness—players must orient themselves within the 3D environment to locate energy cores and navigate to different areas of the village. The audio visualizations respond to the player's position in space, creating a dynamic soundscape that changes based on proximity to different elements. Environmental storytelling is embedded in the 3D layout, with the arrangement of buildings, characters, and cultural exhibits creating a narrative that unfolds through exploration. The scale and depth perception of 3D space communicate the vision of a sustainable community more effectively than any flat representation, allowing players to truly feel immersed in the solarpunk village.

Key Features

  • Interactive 3D exploration with keyboard and voice controls
  • VR headset compatibility with A-Frame
  • Energy core collection gameplay mechanic
  • Dynamic audio visualizations reactive to player position
  • Animated characters and holographic displays
  • Procedurally generated mathematical art exhibits
  • Music-reactive environmental elements
  • Cultural exhibits and environmental narrative

External Project: Probabilities - Mathematical Visualization

About this Project

Probabilities is an interactive mathematical art piece that visualizes complex probability concepts through generative design. The project transforms abstract mathematical formulas into elegant, dynamic visual patterns that users can manipulate in real-time. By adjusting parameters, viewers can see how changes in probability distributions create vastly different visual outcomes.

This project originated from my fascination with the intersection of mathematics, data visualization, and art. I wanted to make probability theory—often seen as dry and abstract—visually compelling and intuitively understandable. The interactive nature allows users to develop an intuitive sense of statistical concepts through play and experimentation.

Key Features

  • Real-time probability distribution visualization
  • Interactive parameter controls
  • Multiple probability models (Normal, Binomial, Poisson)
  • Generative visual patterns based on mathematical logic
  • Educational tooltips and explanations

Technologies Used

  • p5.js
  • JavaScript
  • Mathematical modeling libraries
  • HTML5 Canvas
